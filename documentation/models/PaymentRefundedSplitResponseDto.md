# PaymentRefundedSplitResponseDto

Refunded Splits, if any

**Properties**

| Name  | Type    | Required | Description                              |
| :---- | :------ | :------- | :--------------------------------------- |
| id    | String  | ❌       | Unique split identifier                  |
| value | Double  | ❌       | Refunded value                           |
| done  | Boolean | ❌       | Indicates whether the split was refunded |
